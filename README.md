## Buildpack Information Appliction

### http://bpinfo.stage1.mybluemix.net

* Shows all buildpacks in each target location
* Cached Feature
  * Will update every 90 minutes.
  * Option to manually refresh. This can only be done once every 3 minutes to prevent spamming.
  * Timer to know how old the current cache is.

* Supported Targets:
  * US Dallas Stage 1
  * US Production
  * London Stage 1
  * London Production
